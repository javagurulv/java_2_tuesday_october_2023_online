"Путь в тысячу миль начинается с первого шага" - Лао-цзы.

Ваша задача: заполнить поля ответа, значения для заполнения
взять из запроса. Изменения протестировать с помощью тестов.
(TravelCalculatePremiumServiceImpl)

Для разработки каждого задания вам нужно создавать отдельный branch
в Git. Название branch должно быть: step_x_description.
Например: step_2_fillResponseFields. Branch с произвольными названиями
не будут приниматься.

После окончания разработки задания вы должны создать pull request
из своего branch в main branch и отдать этот pull request на
code review ментору. Только после успешного прохождения code review
и получения approve от ментора вы можете сделать merge своего branch
в main. После pull request merge работа над заданием считается
оконченным если в самом задании не написано, что-то другое.
